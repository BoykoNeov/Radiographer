# M2 — Data pipeline: G-P gamma-ray buildup factors

**Status:** done ✅ (third of M2's §7 datasets, after emissions ✅ and attenuation ✅;
siblings — conversion coefficients, neutron terms, spent-fuel vectors — still to land)
**Milestone (HANDOFF_PLAN.md §10):** assemble the §7 datasets with finalized
schemas, build loaders/interpolators, write regression tests as each table lands.

## Results (done)

- **8 canonical files** `data/buildup/<material>.json` (`schema_version: 1`), generated by
  `data/build/build_buildup.py` from the hand-keyed transcription of NUREG/CR-5740.
- **Materials:** lead, tungsten, iron, copper, aluminium, concrete + water, air.
  PMMA/polyethylene/tissue intentionally absent (no ANS-6.4.3 data).
- **Loader + evaluator** `engine/buildup.py` (`gp_buildup`, `buildup_factor` at grid
  energies; energy interpolation deferred to M3).
- **10 regression tests pass** (`tests/test_buildup_data.py`); full suite **115 pass**,
  no emissions/attenuation/M1 regression.
- **Validation outcome (strong):** reconstructed B reproduces the report's *own* Table 3
  exposure B-values to **≤ 2.6%** across **all 8 materials** (37 anchors, low→high Z) —
  every material has value-level regression coverage, not just the evaluator-only
  algebraic goldens. Iron is additionally cross-checked against an independent
  open-access tabulation (EPJ 2016, CC-BY) at 5 energies. The air anchors come from the
  report's air *exposure* B-table, confirming air's energy-absorption coefficients (the
  only set tabulated) reproduce its exposure B-values (they coincide for the air medium).

> This is the **buildup factor** B(E, x) that corrects the bare point-kernel for
> scattered photons: `I = I₀ · B(E, μx) · e^(−μx)` (§6.5). Omitting B badly
> *under*estimates dose through thick shields. It is the last M3-gating dataset on
> the material side (with attenuation), feeding the M3 gamma dose engine.

## Decisions (LOCKED for this dataset)

### Source — degraded provenance model (vs attenuation/emissions)
- **Primary source: NUREG/CR-5740 / ORNL/RSIC-49 (D. K. Trubey, Aug 1991)**,
  *New Gamma-Ray Buildup Factor Data for Point Kernel Calculations: ANS-6.4.3
  Standard Reference Data*. **Identical data** to the (paywalled, copyrighted)
  ANSI/ANS-6.4.3-1991 standard; this report is a **U.S. DOE/ORNL public-domain**
  document. DOI 10.2172/7032799 (OSTI). Vendored as `data/vendor/ans643/NUREG-CR-5740.pdf`.
- **Why not the verbatim-bytes + hash + independent-re-parse model** used for NIST
  attenuation and ICRP-107 emissions: there is **no clean, machine-readable,
  openly-licensed** ANS-6.4.3 G-P dataset. The machine-readable original (RSICC
  **DLC-129** "ANS643", 5.25″ Fortran card images) is access-gated/likely
  export-controlled and not directly downloadable; the standard itself is paywalled;
  the public-domain NUREG is a **scanned PDF with an unreliable OCR layer** (ABBYY
  FineReader 10 — observed digit errors: `8.6721`, `--0.0071`, dropped `Xk` cells).
- **The coefficients are physical fit-constants (facts, not copyrightable)**, so we
  ship canonical JSON citing NUREG/CR-5740. The trust model **degrades honestly**
  from "re-parse vendored bytes" to **transcription + cross-check** (see Validation).

### Response function
- **Exposure (air-kerma) buildup factor**, point-isotropic source, infinite medium —
  consistent with the μ_en/ρ-in-air dose path (§6.1). The report also tabulates an
  **energy-absorption ("kerma")** response per material; we transcribe **exposure
  only** for v1. (Energy-absorption buildup is a future add if M3/M4 needs it.)

### Scope — materials (resolves the coverage gap honestly)
- Transcribe: **lead, tungsten, iron, copper, aluminium, concrete** (the shields
  ANS-6.4.3 covers) + **water, air**.
- **PMMA, polyethylene ship NO buildup** — ANS-6.4.3 has no data for them. The M3
  dose engine must handle their absence **loudly** (refuse, or a *documented* low-Z
  surrogate), never a silent substitution (CLAUDE.md no-silent-errors).
- **tissue_soft is intentionally absent** — it is a *scoring medium*, never a shield
  you penetrate; no buildup factor is needed for it.

### Energy-boundary contract — DIFFERS from attenuation (M3 carry-forward)
- ANS-6.4.3 grid is **0.015–15 MeV, 25 points** (vs NIST's 1 keV–20 MeV). So the
  attenuation "skip photon lines below 1 keV" rule does **not** cover buildup. M3
  needs a **separate, documented** rule: photon lines **below 15 keV** (B≈1 there —
  photoelectric dominates, negligible scatter) and **above 15 MeV** are handled
  explicitly and logged, never silently extrapolated off the grid.

### Layered-shield buildup — still OPEN (HANDOFF_PLAN §6.5 / §13 #2)
- Whose B for a Pb-then-water stack has no clean theory. Pick a documented
  approximation (last-layer, or Harima–Kitazume) **in M3** and state it in the UI.
  This dataset ships **single-material, infinite-medium** coefficients only.

## Source layout (NUREG/CR-5740, appendix "Table 5.1")
Each material occupies ~one PDF page with two stacked sub-tables: *G-P Energy
Absorption Buildup Factor Coefficients* (kerma) then *G-P Exposure Buildup Factor
Coefficients* (the one we take). Columns: `E(MeV) b c a Xk d | Max.Dev. Xmax St.Dev.`
(the last three are fit-quality metadata, not GP parameters).

Exposure-table PDF pages: aluminium **103**, iron **110**, copper **111**, tungsten
**116**, lead **117**, water **119**, air **120**, concrete **121**.

The report also tabulates the actual **B(E,x) values** (Table 3 *Exposure Buildup
Factors*; Table 4 *Energy Absorption Buildup Factors*) at standard mfp depths — used
for a within-source reconstruction spot-check.

## G-P (geometric-progression) formula (Harima)
```
B(E, x) = 1 + (b − 1)(K^x − 1)/(K − 1)      for K ≠ 1
B(E, x) = 1 + (b − 1)·x                       for K = 1
K(E, x) = c·x^a + d·(tanh(x/Xk − 2) − tanh(−2)) / (1 − tanh(−2))
```
x = penetration depth in **mean free paths (mfp)**; b = buildup at 1 mfp.
Algebraic identities (free goldens): **B(0)=1**, **B(1 mfp)=b** exactly.

## Validation design (LOCKED — advisor-reviewed, proportionate)
Backbone (all):
1. **Double-entry transcription** — image read vs the ABBYY OCR layer; every
   disagreement reconciled by re-inspecting the page image. Sharpest per-number check.
2. **Algebraic goldens** — reconstructed `B(0)=1`, `B(1 mfp)=b`. Free evaluator check.
3. **Cross-source sample** — coefficients vs *independent* open-access ANS-6.4.3
   reproductions (e.g. EPJ Web Conf. 106, 03009 (2016), CC-BY, Table 1 = iron at 7
   energies). A cross-source value can itself be corrupt (that paper's iron@10 MeV
   row is garbled) — on disagreement, **investigate both sides**, plausibility +
   smoothness as tiebreaker; never auto-assume the transcription is wrong.
4. **Smoothness** — GP params vary smoothly in log-E (e.g. water `b`: 1.18→~5→1.27,
   a single hump). Catches the residual double-entry misses (both channels misread
   the same clean glyph).
5. **Reconstruction spot-check** (NOT full Table 3) — transcribe Table 3 B-values for
   **all 8 materials** (~2–3 (E, mfp) points each, 37 total), reconstruct via the GP
   formula, require agreement within the report's tabulated `Max.Dev` + margin. Catches
   **systematic** errors (c↔a column swap, kerma/exposure mixup, formula bug) AND gives
   each material per-material value-level coverage (the algebraic goldens are
   value-independent, so without an anchor a material's c/a/Xk/d have no regression
   coverage). Also validates the evaluator for M3. Coverage of every required material is
   asserted by the test.

**Not** a hard invariant: `b ≥ 1`. Whether exposure `b` dips below 1 at high energy
is verified empirically from NUREG, not assumed — a too-tight bound tempts silent
"fixing" of transcribed values (forbidden). Structural checks assert only loose
sanity (finite, generous range).

**Trust boundary (honesty):** faithfulness to true ANS-6.4.3 is independently verified
only for the cross-source / golden subset; the remainder is trusted to the
double-entered transcription. This mirrors the emissions trust boundary.

## Canonical schema (`schema_version: 1`)
```jsonc
{
  "schema_version": 1,
  "material": "lead",
  "name": "Lead",
  "response": "exposure",                  // air-kerma; point-isotropic, infinite medium
  "source": "ANSI/ANS-6.4.3-1991 via NUREG/CR-5740 (Trubey 1991), G-P exposure buildup",
  "E_MeV": [ 0.015, …, 15.0 ],             // ASCENDING, the 25-pt ANS-6.4.3 grid
  "gp": [ { "b": …, "c": …, "a": …, "Xk": …, "d": … }, … ]   // aligned to E_MeV
}
```
